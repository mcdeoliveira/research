Matlab

We will make extensive use of Matlab for computations, so it is important that you have access to a computer with Matlab.

You will need Matlab to complete some of the assignments.

In this class, almost everything you need can be done with the Student Version of Matlab (100$ at the bookstore) as well as with the free Matlab clone Octave.
