RhythMech

By Allison Hsieh, Shreya Mani, Vivianne Le

RhythMech is an electronic drumstick. Our project will use 3 kinds of software: buttons, a gyroscope, and accelerometer. The gyroscope will turn the machine on and off. If the box is vertical, the machine will not generate any music. However, if the machine is tilted to the side, sound will be generated. The accelerometer will control the volume of the sound. If there is a greater acceleration, the sound will be louder and if it is smaller, the sound will be softer. We are planning to use an MPU6050, which is an inertial measurement unit that contains both a gyroscope and accelerometer. Different combinations of the three buttons being pressed down will generate sounds from different parts of the drum.
