Projects

Projects can have their emphasis in one of four areas:

Live performance
Compose or play a piece using your own instruments or instruments constructed using Little Bits or PD. We will help you incorporate some of the techniques discussed in class using Little Bits, PD and Ableton in the context of a live performance.
Recorded performance
Compose or play a piece using your own instruments or instruments constructed using Little Bits or PD. We will help you incorporate some of the techniques discussed in class using Little Bits, PD and Ableton in the context of a recording session.
Installation
Design and construct a device for a musical installation. We will help you incorporate some of the techniques discussed in class using Little Bits, PD and the Raspberry Pi or Arduino boards. A piece can be composed for the installation.
Technology
Development of musical technologic to construct a new instrument or process audio signals. We will help you incorporate some of the techniques discussed in class using Little Bits, PD and the Raspberry Pi or Arduino boards.
